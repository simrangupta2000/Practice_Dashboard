🛒 Blinkit Sales Analysis – Power BI Dashboard
📊 Project Overview

This project is a Power BI practice dashboard built to analyze sales performance and product insights for Blinkit, India’s last-minute delivery app. The dashboard provides a clear view of sales trends, outlet performance, product distribution, and customer ratings through interactive visualizations.

The objective of this project was to practice data analysis, dashboard design, and interactive reporting using Power BI while transforming raw data into meaningful business insights.

✨ Dashboard Features

✔ Interactive filter panel for Item Type, Outlet Location Type, and Outlet Size
✔ Sales performance overview including average rating, average sales, and total sales
✔ Outlet establishment trend analysis over the years
✔ Fat content distribution to analyze product categories
✔ Outlet size analysis showing sales contribution by outlet size
✔ Outlet location performance comparison across Tier 1, Tier 2, and Tier 3
✔ Outlet type performance table displaying sales, ratings, and number of products

📈 Key Insights

Supermarket Type1 generates the highest total sales among outlet types.
Tier 3 outlet locations contribute significantly to overall sales.
Medium and small outlets show balanced sales distribution.
Average product rating remains consistently around 3.9 across outlets.
